URL Shortener API

Overview

This API converts any URL into a short URL by storing the URL into the database and assigning it an index. The short URL will redirect to the user to the original URL. If no such URL exists, an error will occur. Requests are handled by Express.js, and a NOSQL database is used to store URLs with MongoDB.

If we type a URL, in this case, https://www.google.com, it will send a POST request to store the URL and create an short URL with an assigned counter. The short URL would be look like this: [url]/api/shorturl/1/.
